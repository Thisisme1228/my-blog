## eslint-config-prettier 
> Turns off all rules that are unnecessary or might conflict with Prettier.
```
Add "prettier" to the "extends" array in your .eslintrc.* file. Make sure to put it last, so it gets the chance to override other configs.

{
  "extends": [
    "some-other-config-you-use",
    "prettier"
  ]
}
```

## prettier-plugin-tailwindcss
> A Prettier v3+ plugin for Tailwind CSS v3.0+ that automatically sorts classes based on our recommended class order.
```
make new file named "prettier-plugin-tailwindcss" to root fold, and put below code in it to enable this plugin:
module.exports = {
    plugins: ["prettier-plugin-tailwindcss"],
}
```

## configuring the invalidation period of the client router cache
```
next.config.mjs

const nextConfig = {
    experimental: {
        //staleTimes is an experimental feature that allows configuring the invalidation period of the client router cache.
        staleTimes: {
            dynamic: 30 //The dynamic property is used when the prefetch prop on Link is left unspecified.
        }
    }
};
```
